# ROS2 Random Distance Publisher
## Description
This project implements a basic ROS2 system with two nodes (node1 and node2) that publish randomly
generated distance values to separate topics. It demonstrates fundamental ROS2 concepts such as node
creation, publishers, and topic-based communication using Python (rclpy)



## ⚙️ Installation

```bash
cd ~/ros2_ws
colcon build
source install/setup.bash
ros2 run dis_pub node1
ros2 run dis_pub node2
```
